## Project Setup

```sh

```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### For run server (json datas)
```sh
npm run server
```

### For run app
```sh
yarn dev
```
